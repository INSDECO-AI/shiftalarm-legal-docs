Политика конфиденциальности ShiftAlarm

Версия 1. Дата: 23.06.2026.

1. Локальные данные

ShiftAlarm хранит пользовательские данные преимущественно локально на устройстве: будильники, сменные графики, гео-будильники и координаты выбранных геозон, заметки, напоминания, аудиозаметки, настройки, виджеты, параметры зарплатного расчёта, диагностику и резервные копии, созданные пользователем.

2. Диагностика

Диагностические журналы предназначены для поиска ошибок. Пользователь сам решает, копировать, очищать или отправлять диагностику разработчику. Приложение не должно отправлять диагностические журналы без действия пользователя.

3. Загрузки из сети

Приложение может использовать интернет для загрузки языковых пакетов, моделей распознавания речи, правовых документов и справочных материалов. При таких запросах удалённый сервер может видеть технические данные соединения, например IP-адрес и сведения о запросе.

4. Резервные копии

Резервные копии создаются по действию пользователя. Если пользователь сохраняет или отправляет резервную копию через другие приложения, дальнейшая обработка файла зависит от выбранного приложения или сервиса.

5. Разрешения

Разрешения на уведомления, точные будильники, геолокацию, фоновую геолокацию, микрофон и файлы используются только для соответствующих функций приложения.
