ShiftAlarm Terms of use

Version 1. Date: 2026-06-23.

1. App purpose

ShiftAlarm helps the user plan shifts, alarms, geo-alarms, notes, reminders, timer, stopwatch, widgets and user-data backups.

2. User responsibility

The user is responsible for configuring schedules, alarms, geofences, notifications, permissions, battery restrictions and backups. Before important events, the user should check that the required functions are enabled and working correctly on the specific device.

3. Operating limitations

Alarms, notifications, background services and geofences depend on Android, the device manufacturer, battery settings, location accuracy, permissions, reboots, system updates and other factors that the app cannot fully control.

4. No critical sole-use

ShiftAlarm is not intended to be the only tool for tasks where failure may create risks to life, health, safety, legal status, employment, money or other significant interests. Use additional independent checks for critical tasks.

5. Document changes

Documents may be updated. Later versions of the app can download current documents from a remote source and keep bundled documents as fallback.
