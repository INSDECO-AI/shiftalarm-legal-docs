ShiftAlarm Privacy policy

Version 1. Date: 2026-06-23.

1. Local data

ShiftAlarm stores user data mainly on the device: alarms, shift schedules, geo-alarms and selected geofence coordinates, notes, reminders, audio notes, settings, widgets, salary calculation settings, diagnostics and backups created by the user.

2. Diagnostics

Diagnostic logs are used for troubleshooting. The user decides whether to copy, clear or send diagnostics to the developer. The app should not send diagnostic logs without user action.

3. Network downloads

The app may use the internet to download language packs, speech recognition models, legal documents and help materials. During such requests, the remote server may see technical connection data such as IP address and request metadata.

4. Backups

Backups are created by user action. If the user saves or shares a backup through other apps, further processing depends on the selected app or service.

5. Permissions

Permissions for notifications, exact alarms, location, background location, microphone and files are used only for the corresponding app functions.
